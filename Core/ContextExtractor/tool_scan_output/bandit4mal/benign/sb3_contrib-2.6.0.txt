11	# Stable-Baselines3 - Contrib (SB3-Contrib)
12	
13	Contrib package for [Stable-Baselines3](https://github.com/DLR-RM/stable-baselines3) - Experimental reinforcement learning (RL) code.
14	"sb3-contrib" for short.
15	
16	### What is SB3-Contrib?
17	
18	A place for RL algorithms and tools that are considered experimental, e.g. implementations of the latest publications. Goal is to keep the simplicity, documentation and style of stable-baselines3 but for less matured implementations.
19	
20	### Why create this repository?
21	
22	Over the span of stable-baselines and stable-baselines3, the community has been eager to contribute in form of better logging utilities, environment wrappers, extended support (e.g. different action spaces) and learning algorithms.
23	
24	However sometimes these utilities were too niche to be considered for stable-baselines or proved to be too difficult to integrate well into the existing code without creating a mess. sb3-contrib aims to fix this by not requiring the neatest code integration with existing code and not setting limits on what is too niche: almost everything remotely useful goes!
25	We hope this allows us to provide reliable implementations following stable-baselines usual standards (consistent style, documentation, etc) beyond the relatively small scope of utilities in the main repository.
26	
27	
28	## Features
29	
30	See documentation for the full list of included features.
31	
32	**RL Algorithms**:
33	- [Augmented Random Search (ARS)](https://arxiv.org/abs/1803.07055)
34	- [Quantile Regression DQN (QR-DQN)](https://arxiv.org/abs/1710.10044)
35	- [PPO with invalid action masking (MaskablePPO)](https://arxiv.org/abs/2006.14171)
36	- [PPO with recurrent policy (RecurrentPPO aka PPO LSTM)](https://ppo-details.cleanrl.dev//2021/11/05/ppo-implementation-details/)
37	- [Truncated Quantile Critics (TQC)](https://arxiv.org/abs/2005.04269)
38	- [Trust Region Policy Optimization (TRPO)](https://arxiv.org/abs/1502.05477)
39	- [Batch Normalization in Deep Reinforcement Learning (CrossQ)](https://openreview.net/forum?id=PczQtTsTIX)
40	
41	**Gym Wrappers**:
42	- [Time Feature Wrapper](https://arxiv.org/abs/1712.00378)
43	
44	## Documentation

47	
48	
49	## Installation
